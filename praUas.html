Data Preprocessing#

Pada Tahapan berikutnya adalah memproses data yang kita punya menjadi suatu dataset yang baik agar ketika penulis membuat modelnya menjadi tidak error. Kolom yang memiliki missing value nantinya akan penulis ganti dengan data lain, mungkin akan diganti dengan menggunakan data mean pada kolom tersebut atau bahkan modus. Perlu diingat juga bahwa penulis tidak akan memasukkan semua fitur kolom untuk dianalisa, karena tentu saja nanti akan tidak relevan dengan apa yang ingin dituju. Jika memasukkan data yang tidak relevan dengan model bisa saja yang dihasilkan oleh model tersebut akan jelek atau bahkan sampah.

Sebelum melakukan data preprocessing, perlu di ingat bahwa sebisa mungkin jangan pernah menyentuh data asli. Hal ini dilakukan supaya ketika kita menganalisa datanya dan ada programnya salah maka kita masih bisa kembali ke data aslinya, tanpa melakukan hal hal yang ribet.

df_copy = df.copy()

Deteksi Missing Value#

Pada tahap data preprocessing penulis akan mengidentifikasi type data pada kolom yang ada pada dataset terlebih dahulu. Identifikasi ini diperlukan agar program yang nanti dijalankan tidak salah mengganti data missing valuenya. Setelahnya penulis merubah dataset yang awalnya merupakan missing value menjadi ada, agar nanti ketika sudah diproses dan dijadikan model muncul hasil yang lebih baik. berikut ini adalah program yang akan mengganti missing valuenya.

# Daftar kolom numerik berdasarkan informasi dari kamu
numeric_cols = [
    'Age', 'Number of sexual partners', 'First sexual intercourse', 'Num of pregnancies',
    'Hormonal Contraceptives (years)', 'IUD (years)', 'STDs (number)',
    'STDs: Number of diagnosis', 'STDs: Time since first diagnosis', 'STDs: Time since last diagnosis'
]

# Kolom target yang tidak akan diimputasi
target_cols = ['Biopsy', 'Hinselmann', 'Schiller', 'Cytology']

# Sisanya kita anggap biner (berdasarkan bool di daftar kamu)
binary_cols = [col for col in df_copy.columns if col not in numeric_cols + target_cols]

# Imputasi numerik: isi dengan mean
for col in numeric_cols:
    mean_val = df_copy[col].mean()
    df_copy[col].fillna(mean_val, inplace=True)

# Imputasi biner: isi dengan modus
for col in binary_cols:
    mode_val = df_copy[col].mode()[0]
    df_copy[col].fillna(mode_val, inplace=True)

# Cek hasilnya
print("Total missing value setelah imputasi:", df_copy.isnull().sum().sum())

SMOTE(Synthetic Minority Over-sampling Technique)#

Pada pembuatan model Decision Tree saya menggunakan metode SMOTE dikarenakan data yang saya miliki dominan pada sisi tertentu saja. Metode Smote ini juga akan saya lakukan bukan saat proses Data Preprocessing, karena jika dilakukan pada saat proses tersebut, akan terjadi kebocoran data yang akan menyebabkan rusaknya data yang ingin diproses. Proses SMOTE hanya dilakukan pada saat training Model nya saja tanpa menyentuh data aslinya.
